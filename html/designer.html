Featured Designer: Chris Coyier

Early Life
Chris Coyier is a web designer and developer who was born in August of 1980. He began his work in web design by starting work at American Family Insurance (software testing) in July of 1998. After this first job, he followed it up with jobs at American (prepress & print design), Suttle-Strauss (prepress), and Chatman Design (web design). One month after starting work at Chatman Design as a web designer, he launched his first company of his own (CSS-Tricks).
His Work
Chris Coyier is most notable for creating CSS-Tricks which is a website based on web design as well as CodePen which is defined as an “a social development environment for front-end web designers and developers”. Which is a community devoted to coding and allows for networking, job search, code sharing, etc. almost as if it is a mixture of both a social network and a blog/forum. Another activity that Chris is involved in would be that he co-hosts a podcast along with a man named Dave Rupert which is centered around building websites and has been going for 6 years with over 300 episodes. All of Chris’ hobbies revolve around building and designing websites which is why I picked him as my featured designer. His life revolves around web design and involved both the front-end and back-end of website design. With such a passion for web design, he frequently goes and speaks at conferences such as An Event Apart, Smashing Conf, and Webstock. Along with creating a blog, owning a web design community platform, hosting a podcast, and speaking at conferences, he also finds time to have written two books on the topic of web design. These books are called Practical SVG and Digging into WordPress. Which have to do with using SVG for front-end design and how to work in WordPress for website development respectively. There are various topics that Chris covers while he jumps around speaking at different conferences, these include CSS, HTML, JavaScript, SVG, web design and development. He also speaks about how to run a web business. With a web business of his own, Coyier is very experienced in starting and running a web business. His website codepen.io is a very fun and interactive resource for fellow web designers and developers because it allows the sharing of code as well as showcasing countless websites with various approaches to the code itself. The website allows you to do it all, you can “...build and deploy a website, show off your work, build test cases, and find inspiration” (codepen.io). There are also various categories where work is shown throughout the home page of the site, to allow other users to view the work. Along with aiding in the development and showcasing of websites, codepen also helps educate its users with various readings and even teaches the basics of web design as well as using real-time viewing so that a beginner can see how code can be used to build a front-end design.
